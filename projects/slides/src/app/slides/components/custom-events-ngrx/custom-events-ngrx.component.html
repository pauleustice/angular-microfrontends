Custom events
  • The most basic form of communication
  • Not suitable for enterprise or large applications
Host (or consuming remote)
window.addEventListener('updateTitle', (e: CustomEvent) => {{ '{' }}
  this.title = e.detail.title;
});
Remote
window.dispatchEvent(
  new CustomEvent('updateTitle', {{ '{' }}
    detail: {{ '{' }}
      title: 'iPhone',
    },
  }),
);
NgRx
  • Used in this repo!
  • You'll need a shared library within the monorepo to use exported actions
this.store.dispatch({{ '{' }} type: '[UI] Update Title', title: 'iPhone' });
