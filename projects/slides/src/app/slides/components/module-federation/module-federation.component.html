Overview
  • New in Webpack 5
  • Allows remotely deployed applications to work together to form a holistic application
  • ...by sharing their inner modules to consumers
  • Federated modules can define shared dependencies, to avoid downloading large libraries multiple times
  • Each consumed module can also consume other federated modules
Federation?
“Federation in a general sense, refers to a union of partially self-governing entities sitting under a central governor.”

Module Federation, then, can be seen as a central application (the host) that can load other modules (remote entities). Federated modules are deployed remotely, and independent of the host, but can be loaded dynamically into any another application as long as it supports it.
